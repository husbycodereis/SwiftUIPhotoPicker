# SwiftUIPhotoPicker

Photo picker app that has access to camera and photo library

The project feature:

* access the camera and photo library
* take pictures
* save pictures 
* cache pictures to local directory
* delete / edit pictures
* error handling
* view the saved pictures in a responsive layout
